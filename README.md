# Talleres_Semillero
El script Shell permite :
1) Cambiar nombre Servidor
2) Cambiar Particion Discos
3) Cambiar IP Servidor
4) Cambiar tabla de host
5) Agregar Permisos de Firewall
6) Editar DNS server
7) Configurar proxy
